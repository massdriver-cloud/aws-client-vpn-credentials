# Operator Guide

## Setup

### Install a client

- [AWS Client VPN for Windows](https://docs.aws.amazon.com/vpn/latest/clientvpn-user/client-vpn-connect-windows.html)
- [AWS Client VPN for macOS](https://docs.aws.amazon.com/vpn/latest/clientvpn-user/client-vpn-connect-macos.html)
- [AWS Client VPN for Linux](https://docs.aws.amazon.com/vpn/latest/clientvpn-user/client-vpn-connect-linux.html)
- [OpenVPN Connect](https://openvpn.net/client/)

### Download the Client Config

- Download the Client Config artifact (_Optionally, you can download the CA certificate, client certificate, and client private key_)
- Load `.ovpn` file into your preferred VPN client (or use the 3 certificates)
- Connect
